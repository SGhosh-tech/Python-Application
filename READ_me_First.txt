1.  Before running the program , make sure you have python installed in your system. If not, download and install it from the official python website:    https://www.python.org/downloads/

2.  Make sure you have the necessary packages installed. You can install them using pip, which are there in the py_app/requirements.txt
You can install all of them collectively by running the following command in your terminal:    pip install -r py_app/requirements.txt

3.  Run the program by executing the following command in your terminal:    python py_app/pass_utility.py

4.  For first time login purpose register with a new username and password. After that you can login with your registered username and password.

5.  Finally u r there to use the application.   (Further modifications are warmly welcomed)
